# 默写单词 VSCode 插件

一个专注于英语单词默写练习的 VSCode 插件，支持多种词典，帮助你在编码之余提升英语词汇量。

## 功能特性

- ✅ 多词典支持（CET-4、CET-6、小学、初中、高考等）
- ✅ 实时拼写检查，输入正确自动标记
- ✅ 学习进度保存，下次打开继续练习
- ✅ 随机排序功能，打乱单词顺序
- ✅ 分页显示，每页 10 个单词
- ✅ 统计显示已掌握和剩余单词数量
- ✅ 完美适配 VSCode 主题

## 词典

- **CET-4**: 大学英语四级 (2607 词)
- **CET-6**: 大学英语六级 (2345 词)
- **小学英语**: 小学英语大纲 (532 词)
- **初中英语**: 初中英语大纲 (1674 词)
- **高考 3500**: 高考核心词汇
- 等等
