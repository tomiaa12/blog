# 后台管理模板

## 站长写的 Vue3 后台管理仓库

Vite4 + Vue3 + Element Plus + Typescript 后台管理系统

- 解决了左侧菜单与顶部 tag 的动态路由切换+缓存的问题
- 解决了全局的`layout`中`transion`组件与`keep-alive`并存且页面不存在根标签时，切换页面展示`animate.css`的动画

发现很多后台管理模板项目都没有解决以上问题，所以自己写一个
