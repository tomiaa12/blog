# 数组

## 声明数组

声明：数据类型 变量名\[元素个数\] = \{ 值 1, 值 2 \}

取值：变量名[下标] 下标从 0 开始

```c
#include <stdio.h>

int main(void)
{
	int a[4] = { 11, 233, 2, 3};

	printf("%d",a[0]);

  // 声明数组时开辟的是连续的内存空间，查看数组每一项在内存中的地址，可以发现每一项占 4 个字节
	printf("%p\n",&a[0]);
	printf("%p\n",&a[1]);
	printf("%p\n",&a[2]);
	printf("%p\n",&a[3]);
}
```
